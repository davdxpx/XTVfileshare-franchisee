# Security Policy

## Supported Versions

Only the latest version of the XTV Fileshare Bot Franchisee Edition is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please do **NOT** open a public issue.

Instead, please report it directly to the developer:

*   **Telegram:** [@davdxpx](https://t.me/davdxpx)

Please include:
*   A description of the vulnerability.
*   Steps to reproduce the issue.
*   Any potential impact.

We will acknowledge your report and work to resolve the issue as quickly as possible. We appreciate your responsible disclosure and cooperation in keeping the XTV ecosystem secure.
