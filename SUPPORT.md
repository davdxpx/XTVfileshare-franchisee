# Support for XTV Fileshare Bot - Franchisee Edition

Need help with your Franchisee setup? We're here to assist you.

## How do I get a Franchisee ID?

To run the Franchisee Edition, you need a valid `FRANCHISEE_ID` and `FRANCHISEE_PASSWORD`, along with connection details for the `MAIN_URI` and `USER_URI` databases.

These credentials are **strictly controlled** and issued only to authorized partners.

*   **Contact:** [@davdxpx](https://t.me/davdxpx) on Telegram.
*   **Subject:** Request Franchisee Credentials.

## Where can I get help?

If you are an existing Franchisee and need technical support, bug fixes, or have feature requests:

1.  **Direct Support:** Contact the developer directly on Telegram at [@davdxpx](https://t.me/davdxpx).
2.  **Documentation:** Please review the `README.md` for installation and configuration steps.
3.  **Community:** Join the official XTV community (link provided upon registration) for updates and announcements.

## Reporting Issues

If you believe you have found a bug in the code:

*   Check existing issues to see if it has already been reported.
*   If not, open a new issue with a clear title and description.
*   **Security Vulnerabilities:** Do NOT open a public issue. See `SECURITY.md` for instructions.

**Note:** Support is provided on a best-effort basis for the community. Priority support is available for licensed partners.
